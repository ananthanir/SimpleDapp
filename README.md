# SampleDapp

## Steps to Execute App
1. Install MetaMask
2. Code the contract, connect Remix to MetaMask.
3. Deploy to Ganache or Testnet.
4. Copy Contract Address and paste it to application (Line No: 38, File: app,js).
5. Run: `npm install`
6. Run: `npm start`

## Execution Flow
1. Connect to MetaMask
2. Store 
3. Get Value
